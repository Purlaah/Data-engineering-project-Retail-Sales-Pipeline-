# Retail Sales ETL Pipeline

This project is a beginner data engineering project where I’m building a retail sales ETL pipeline using Python and pandas.

The goal is to simulate a real-world workflow where raw transactional sales data is cleaned, processed, and prepared for analysis/dashboarding.

The dataset contains intentionally messy retail sales data with:
- missing values
- inconsistent formatting
- invalid entries
- incomplete transaction information

This project is helping me learn more about:
- data cleaning
- ETL processes
- automation
- handling large datasets
- building structured data workflows

---

## Current Progress

- Created project structure
- Loaded raw retail sales dataset into Python
- Built initial ETL pipeline script
- Added dataset validation checks
- Created automatic raw data backup process
- SQLite integration to store clean data into a SQL database
- Windows Task Scheduler for daily automated pipelining
